# Variable: boxStyles

> `const` **boxStyles**: [`Boxes`](../interfaces/Boxes.md)

Predefined box styles for use in the terminal.

## Example

```typescript
import { boxStyles } from "./box-styles";

console.log(boxStyles.single);
// {
//   topLeft: '┌',
//   top: '─',
//   topRight: '┐',
//   right: '│',
//   bottomRight: '┘',
//   bottom: '─',
//   bottomLeft: '└',
//   left: '│'
// }
```
